GMaps.js — Static map with polyline

A single polyline can be also drawed in a static map:

url = GMaps.staticMapURL({
  size: [610, 350],
  lat: -12.043333,
  lng: -77.028333,

  polyline: {
    path: path,
    strokeColor: '#131540',
    strokeOpacity: 0.6,
    strokeWeight: 6
  }
});

$('<img/>').attr('src', url)
  .appendTo('#map');
